객체 생성 방식 3가지
    1. 객체 리터럴 방식
    2. 생성자 함수
    3. Object.create(프로토타입) : 프로토타입 상속을 통해 객체를 만드는 방식(프로토타입, 상속 등의 개념이 필요)

    프로퍼티 / 메소드 접근 방법
    1. 객체.key : 마침표(Dot Notation)로 프로퍼티에 접근
    2. 객체['key'] : 대괄호 사이에 키 값을 '문자열'로 넣어 접근

    객체[key] 방식은 key가 따옴표로 감싸있지 않으면 변수로 해석하여 참조한다
    let key_age = 'age';
    myObj[key_age];
