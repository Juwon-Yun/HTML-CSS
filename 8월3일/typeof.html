typeof : 변수의 데이터 타입을 반환하는 연산자

    원시타입 (primaitive data type)
    undefined : 변수가 정의되지 않았거나 값이 없을 때
    null      : 의도적으로 값이 없음을 명시하기 위해
    number    : 타입이 수 일때
    String    : 타입이 문자열 일때
    boolean   : 타입이 true, false 일때
    object, function : 타입이 함수, 배열 등 객체일 때
    symbol    : 타입이 심볼일 때
        ㄴES6에서 새롭게 추가된 7번째 타입을
            - 변경 불가한 원시 타입의 값으로 다른 값에 중복되지 않는 유일무이한 값
            - 이름의 충돌 위험이 없는 유일한 객체의 프로퍼티 키를 만들기 위해 사용
