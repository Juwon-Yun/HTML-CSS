변수 선언 키워드 3가지
    1. var (함수 범위 : 함수안에 선언된 변수는 그 함수 안에서 유효)
        - 변수 선언과 초기화(메모리 할당)가 동시에 이루어진다.
        - 변수를 재선언해도 에러가 나지 않으며, 각기 다른 값을 출력한다.
        - 유연한 변수 선언 방식으로 의도치 않는 경우가 발생할 가능성이 많다.
    
    때문에 이를 보완하고자 ES2015(ES6버전)이후 let, const 변수 선언 방식이 등장했다.

    2. let (블록범위 : 변수가 선언된 블록, 구문, 표현식 내에서만 유효)
        - 변수 사용 전까지 초기화되지 않는다.
        - 변수 재할당은 가능
        - 변수 재선언은 불가
        - 호이스팅 불가

    3. const (블록범위)
        - 변수 사용 전까지 초기화되지 않는다.
        - 변수의 재할당과 재선언 모두 불가
        - 처음 선언시 반드시 초기화(할당)해야 한다. (초기화 누락 에러가 발생)
        - 보통 대문자를 사용해 선언하지만 강제하지 않는다.

    var과 달리 let, const는 불변하다.

    전역변수(global variable) 와 지역변수(local variable)

    1. 전역변수 : 전역범위의 변수는 코드 블럭의 바깥쪽에서 선언된다.
                 전역변수는 코드 블럭을 포함한 프로그램의 모든 곳에서 접근할 수 있다.
    2. 지역변수 : 코드 블럭 범위안에서 정의된 변수로 그 블럭 안에서만 사용 가능하다.

    *자바스크립트의 전역범위는 함수범위 내부와 동일하게 동작한다.
