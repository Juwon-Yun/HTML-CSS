html을 배웁니다.

css를 이용해서 스타일을 지정합니다.

javaScript를 이용하여 클라이언트와 서버 사이의 상호작용을 합니다

	
	스타일 지정
	
	1.외부 스타일 : 별도의 css파일을 작성하고 저장한 후,
			     html문서 내부에서 링크하여 사용한다
	2.내부 스타일 : html 문서 내부에 
				 안쪽에 작성한다.
	3.인라인 스타일 : 태그 코드 안쪽에 입력하지만 표준 에서 권장하지 않는다
	
	br 태그: 줄 바꿈 (line break)
	
	p 태그 : 단락 지정
	
	pre 태그 : 개발자가 입력하는 그대로 화면에 표시
	
	h1 ~ h6 태그 : headline에 사용하는 태그
	
	ol, ul, dl 태그 : 항목들을 나열하는 리스트
	
	a 태그 : 링크한 문서나 이미지로 이동 
		- href : 링크될 경로 설정
		- target : 링크되는 곳이 열릴 곳 지정
		
	hr태그 : 수평선 태그
	
	img 태그 : 이미지를 표현할 때 사용
	    - src : 파일 경로 지정
	    - alt : 어떠한 이유로 이미지를 표현하지 못할 때
	    		대체되는 텍스트(alternate text)
